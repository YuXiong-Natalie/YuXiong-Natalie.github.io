Data Projects

Riverbed Technology, Data Scientist-part time.

My work in Riverbed Technology involves collaborating with different product teams and solving various data-related problems. I have done mainly two lines of work: 1) data exploration, model build and model validation on NPM (Network Performance Monitoring) metrics; 2) survey design and sentiment analysis on Aternity customer data

NPM metrics

Network health monitoring relies on many diagnostic metrics, such as utilization, throughput, etcs. To automate anomaly detection via using data from multiple source of data, I managed to pull out data from three different network monitoring products through RESTful API and merged them to a cloud-based datastore(Azure). To validate data quality, I used KQL (Kusto Query Language) quering data from a dozen customers with the size each of millions data points and developed algorithm for inspecting missing patterns, extreme values and trends. For time series data such as throughput, we used Holt-Winter exponential smoothing to predict peaks and drops.For multiple metrics of response time, I used unsupervised learning algorithm (e.g. K-means) to find clusters of possible unhealthy response time compositions so as to provide insights for next step analysis

Sentiment Survey

To understand how customers perceive their IT service quality and find out what are the key drivers of the disatisfaction, I worked with Aternity team to design survey questions and implement data analysis. Work includes dashboard design, analytic template building and pilot testing
